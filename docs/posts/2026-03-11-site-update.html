Site Update: Archives and Tags

xq on 2026-03-11

Today I've added several new features to the blog to make it easier to browse and discover content:

New Features

  1. Month-based Archives: You can now browse all posts grouped by month. This is perfect for looking back at older entries by date.
  2. Tag Index: A new "Tags" page serves as a central hub for all post tags, making it easy to find topics you're interested in.
  3. Improved Navigation: I've added direct links to "Archives" and "Tags" in the main navigation menu at the top of every page.

Technical Improvements

  • Absolute URL Paths: Fixed a bug where links would break when navigating from subdirectories. All internal links now use absolute paths.
  • Tag Trimming: Leading spaces in tags are now automatically handled, ensuring clean URLs and correctly grouped tags.

Check out the new browsing options in the menu above!
